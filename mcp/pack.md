Pack a zip file or unzipped directory into a Chrome extension (.crx file).

<usage>
Use this tool when the user wants to create a .crx file from a zip archive or unpacked extension directory. The tool signs the extension with a private key and outputs a proper Chrome extension package that can be installed in Chrome browser.
</usage>

<example>
- "Pack this unpacked extension to a .crx file"
- "Create a crx from ./my-extension/"
- "Pack myextension.zip to ./release/my-extension.crx"
</example>
