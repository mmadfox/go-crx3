Create a zip archive from a directory or file.

<usage>
Use this tool when the user wants to create a zip archive from a directory or individual files. This is useful for preparing extension source files before packing them into a .crx file, or for general compression needs.
</usage>

<example>
- "Zip this directory for packaging"
- "Create a zip archive of the extension source"
- "Compress my-extension/ to my-extension.zip"
- "I need to archive the files before sending them"
</example>
