Extracts the contents of a ZIP archive (e.g., a downloaded Chrome extension or any .zip file) into a directory structure for inspection or modification.

<usage>
Use this tool when the user wants to extract files from a ZIP archive. The tool decompresses the archive into a specified directory, making its contents (such as manifest files, scripts, and assets) accessible for review, analysis, or editing.
</usage>

<example>
- "Extract the contents of archive.zip to ./extracted/"
- "Unzip the downloaded extension to ./source/"
- "I need to inspect the files inside my_project.zip"
- "Decompress data.zip into the current directory"
</example>
